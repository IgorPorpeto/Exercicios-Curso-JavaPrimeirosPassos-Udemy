# Java: Primeiros Passos & Lógica de Programação 🚀
#### Prof. Dr. Nelio Alves | Udemy

Este repositório contém a resolução de todos os exercícios propostos que fiz do curso de **Lógica de Programação** com Java. Este treinamento equivale à disciplina inicial de algoritmos em cursos superiores de Computação.

## 🎯 Objetivo do Curso
Desenvolver o raciocínio lógico e a capacidade de elaborar soluções para problemas computacionais, utilizando Java como ferramenta para implementar fundamentos universais de programação.

## 🧠 Conteúdo Programático
O aprendizado foi estruturado em módulos progressivos, acompanhados de exercícios de fixação e desafios:

*   **Introdução à Programação:** Conceitos básicos e ambiente de desenvolvimento.
*   **Expressões:** Prática com expressões Aritméticas, Comparativas e Lógicas.
*   **Estrutura Sequencial:** Entrada de dados (leitura), processamento (atribuição) e saída de dados (escrita).
*   **Estrutura Condicional:** Controle de fluxo com `if/else`.
*   **Estruturas Repetitivas:** Domínio de laços `while`, `for` e `do-while`.
*   **Estruturas de Dados:** Manipulação de **Vetores** e **Matrizes**.

## 🛠️ Ferramentas Utilizadas
*   **Linguagem:** Java
*   **IDE:** Eclipse (Configuração, Depuração e Execução)
*   **Técnica:** Testes de mesa manuais e esquemas ilustrativos para validação de lógica.

## 📈 Resultados de Aprendizado
- Habilidade para decompor problemas reais em passos lógicos.
- Domínio da sintaxe básica da linguagem Java.
- Base sólida concluída para avançar para o curso "Java COMPLETO" (Orientação a Objetos).
